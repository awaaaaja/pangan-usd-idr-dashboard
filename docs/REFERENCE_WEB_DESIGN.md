# Referensi Desain Web Publik

Referensi ini digunakan untuk **mempelajari pola**, bukan menyalin identitas merek, aset, kode, atau layout secara 1:1.

## 1. Our World in Data
- URL: https://ourworldindata.org/
- Pelajaran: research-first, chart-led, metadata/sumber kuat, eksplorasi topik yang bersih.
- Terapkan: caption sumber di bawah setiap figure, visual lebar, narasi pendek sebelum chart, link artefak.
- Hindari: meniru branding Oxford/OWID.

## 2. Flourish Stories & Examples
- URL: https://flourish.studio/examples/
- Pelajaran: scrollytelling, interactive chart, captions sebagai bagian cerita, transisi antarstate.
- Terapkan: section reveal, storytelling sequence, before/after ablation, interactive evidence drawer.
- Hindari: menyalin template atau visual Flourish secara mentah.

## 3. The Pudding
- URL: https://pudding.cool/
- Pelajaran: visual essay, satu pertanyaan kuat, ilustrasi khas, pengalaman scroll yang memorable.
- Terapkan: hero berani, ilustrasi custom SVG, section dengan ritme berbeda, interpretasi sederhana.
- Hindari: menyalin gaya editorial spesifik atau karya mereka.

## 4. D3 by Observable
- URL: https://d3js.org/
- Pelajaran: bespoke data visualization, transitions, zoom/brush/drag, bentuk visual khusus.
- Terapkan: chart motion ringan, hover/tooltip bermakna, line draw, legend interaktif yang jujur.
- Hindari: mengadopsi D3 bila chart framework yang ada sudah cukup; jangan refactor backend demi efek visual.

## 5. Gapminder Tools
- URL: https://www.gapminder.org/tools/
- Pelajaran: eksplorasi data yang mengajar pengguna, control yang intuitif, indikator waktu.
- Terapkan: explainers singkat, interaksi yang membantu sidang/dosen memahami hasil.
- Hindari: membuat bubble chart bila tidak didukung data.

## Ringkasan arah

Gunakan Our World in Data untuk kredibilitas, Flourish untuk struktur cerita, The Pudding untuk rasa visual, D3 untuk detail interaksi, dan Gapminder untuk pola eksplorasi. Hasil akhirnya harus tetap orisinal dan berbasis data penelitian sendiri.
